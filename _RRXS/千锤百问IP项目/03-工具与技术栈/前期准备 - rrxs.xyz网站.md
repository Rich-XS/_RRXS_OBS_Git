# 前期准备 - rrxs.xyz网站

## 子笔记概述
网站作为IP项目门户，提供内容展示、社群入口。使用腾讯云开发，低成本部署。

标签：#工具 #网站 #前期

## 域名备案步骤
1. 登录腾讯云控制台，进入域名管理。
2. 提交备案申请：主体信息（个人/企业）、网站内容审核。
3. 上传身份证/营业执照，预计7-20天审核通过。
4. 解析域名到云开发环境（CNAME记录）。
5. 测试上线：HTTPS证书免费申请。

注意：备案仅限大陆域名，非大陆可跳过，使用Cloudflare CDN。

## 腾讯云开发教程链接
- 官方文档：https://cloud.tencent.com/document/product/876
- 快速启动：创建云开发项目 → 部署静态网站（HTML/JS）。
- 集成：API网关调用混元AI，数据库存储用户数据。
- 成本：免费额度内<300元/月，超出按量付费。
